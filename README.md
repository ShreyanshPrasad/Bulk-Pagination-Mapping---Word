# Bulk-Pagination-Mapping---Word
This script can be used to create index, pagination and related thing, where continuous page number is required. Using this script we can get start and end page number in continuous order, it can process fils in bulk

---
## Configuration
1. Change the input path
2. Open Excel -> Macros, Shortcut Alt + F8
3. Paste the code
4. Click on run, or hit F5
5. A table will be created in excel, with file name, start page, end page as columns, and one by one data will be filled in.
